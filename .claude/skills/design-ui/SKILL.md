# UI Design & Redesign

Design clean, modern, and usable interfaces.

---

## When to Apply

- Designing new UI
- Redesigning existing pages
- Improving layout and UX
- Working with TailwindCSS, shadcn/ui, framer-motion

---

## Core Principles

- Clarity over creativity
- Simplicity over decoration
- Strong visual hierarchy
- Consistency across UI

---

## Rules

### 1. Layout

- Use clear sections
- Avoid clutter
- Maintain spacing and alignment

---

### 2. Components

- Use reusable UI components or pre-made ui components from popular tailwind or shacdn ui libraries like: Aceternity UI, TailwindPlus, React Bits (more info about other shacdn ui libraries can be found here: https://ui.shadcn.com/docs/directory or check the https://ui.shadcn.com/docs/directory.md )
- Follow consistent patterns
- Avoid unnecessary variations

---

### 3. Typography & Hierarchy

- Clear heading structure
- Proper spacing between elements
- Emphasize important content

---

### 4. UX

- Make actions obvious
- Reduce friction
- Provide feedback for interactions

---

### 5. Motion (Framer Motion)

- Use subtle animations
- Avoid over-animation
- Enhance UX, not distract

---

## Output Requirements

- Layout structure
- Component suggestions
- UX improvements
- Clear reasoning
